\37. List和Set集合有什么区别

List集合中元素有顺序，允许重复。常用的实现类ArrayList，使用List.add()添加元素，使用List.size()获取长度，List.get(下标)获取元素。

Set集合中元素没有顺序，不允许重复

常用的实现类HashSet，使用Set.add()添加元素

\38. List和map集合有什么区别

map储存一组成对的键-值对象（key-value），key不能重复，map.put()添加元素，map.get()获取元素。

不同包且不是子类可以访问的：public
